# ✨ Random-Quote-Generator-Using-Owl
# ✨ QuoteVerse

A modern random quote generator built with **OWL (Odoo Web Library)**, **JavaScript ES6**, **HTML5**, **CSS3**, and **Bootstrap 5**.

The application delivers inspirational quotes through a beautiful glassmorphism interface with dynamic animated backgrounds and a smooth user experience.

---

## 🚀 Features

### 🎲 Random Quote Generation

Generate inspirational quotes instantly with a single click.

### 🌐 API Integration

Fetches random quotes from a public API in real time.

### 🔄 Fallback System

If the API is unavailable, the application automatically displays quotes from a local collection.

### 📋 Copy Quote

Copy the current quote and author to the clipboard with one click.

### 🎨 Dynamic Backgrounds

The background changes automatically with beautiful animated gradients whenever a new quote is generated.

### ⚡ Loading States

Displays loading indicators while fetching quotes for a smooth user experience.

### ✨ Modern UI

- Glassmorphism design
- Animated gradients
- Smooth transitions
- Responsive layout
- Mobile-friendly interface
